# Visual Studio工程生成器

## 这是什么

Visual Studio工程生成器，支持iar、keil和MCUXpresso IDE的工程转换成vs工程，从而利用vs的代码编辑功能加快嵌入式开发。

## 怎么使用

用EmbVSProjectGenerator打开工程配置文件即可
